# Assets

Place Morgana logo files here:

| File | Usage |
|------|-------|
| `logo.svg` | Preferred — scalable, used in UI navbar and docs |
| `logo.png` | Fallback raster version |
| `logo-dark.svg` | Dark background variant (optional) |
| `favicon.ico` | Browser tab icon (place in `ui/` root, not here) |
| `favicon.png` | 32x32 PNG favicon alternative |
